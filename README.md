# 🥗 Учёт калорий — Calorie Tracker

**Версия:** 1.0.2  
**Язык:** Python 3.12  
**Хранение данных:** CSV (встроенный модуль `csv`)  
**Тестирование:** pytest + pytest-cov

---

## Описание

Консольное приложение для учёта съеденных продуктов и расчёта дневной нормы калорий.
Реализует полный цикл разработки: от ТЗ и TDD до приёмочного тестирования и поддержки.

## Инструкция по запуску

```bash
# 1. Клонировать репозиторий
git clone https://github.com/your-username/calorie-tracker.git
cd calorie-tracker

# 2. Создать виртуальное окружение (рекомендуется)
python -m venv venv
venv\Scripts\activate   # Windows
source venv/bin/activate  # Linux/Mac

# 3. Установить зависимости
pip install pytest pytest-cov

# 4. Запустить приложение
python menu.py

# 5. Запустить тесты
pytest tests/ -v

# 6. Проверить покрытие кода
pytest tests/ --cov=. --cov-report=term-missing
```

## Расчёт нормы калорий

Используется формула **Миффлина–Сан Жеора**:

- **Мужчины:** BMR = 10×вес + 6.25×рост − 5×возраст + 5
- **Женщины:** BMR = 10×вес + 6.25×рост − 5×возраст − 161
- TDEE = BMR × коэффициент активности
- Норма = TDEE ± цель (±500 ккал)

## FAQ

**Q: Как сбросить все данные?**  
Удалите файл `data/food_log.csv` или выполните:
```python
python -c "import data_service; data_service.clear_all()"
```

**Q: Как ввести дробный вес?**  
Можно использовать как точку (`100.5`), так и запятую (`100,5`) — оба варианта поддерживаются.
